# Introduction to JavaScript — 3 Mini Projects

## What this bundle REALLY covers (with proof in the projects)

### 1 JavaScript syntax basics

You will _see and use_:

- **Semicolons** `;` (included in examples, and you’ll write them)
- **Parentheses** `()` (function calls like `console.log(...)` and `if (condition)`)
- **Curly braces** `{}` (code blocks in `if/else` and in `@media`-like style blocks, but here JS blocks)
- **Strings** `"text"` and `'text'`
- **Numbers** `520`, `2`, `0.15`, etc.

### 2 Variables + data types

You will practice:

- `const` and `let`
- **string**, **number**, **boolean**

### 3 Operators (all requested are included now)

- Math: `+`, `-`, `*`, `/`, `%`
- Comparisons: `===`, `!==`, `>`, `<`
- Logical: `&&`, `||`

### 4 Commenting + console debugging

- You will add comments and use `console.log()` as a “debug flashlight”
- Every project forces you to read the console output and confirm each step

### 5 Percent / percentage logic

- In Project 2 you will compute a tip using **percent to decimal conversion**
  - ex: `15%` → `15 / 100` → `0.15`

## How to run

1. Open a project’s `MENTEE/index.html`
2. DevTools → Console
3. Follow steps in `MENTEE/script.js`
