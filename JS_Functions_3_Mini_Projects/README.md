---

## Topics Covered

### Declaring and Calling Functions

You will practice:

- Creating a function
- Calling (running) the function

**Why this matters:**  
Functions allow you to write code once and reuse it, instead of repeating the same logic everywhere.

---

### Functions with Multiple Parameters

You will learn how to:

- Pass more than one value into a function
- Use parameters as inputs

**Why this matters:**  
Parameters make functions flexible. The same function can produce different results depending on the values provided.

---

### Arrow Functions and Function Expressions

You will explore:

- Traditional function declarations
- Function expressions
- Arrow functions (`=>`)

**Why this matters:**  
Arrow functions are heavily used in modern JavaScript and frameworks like React. Understanding them early makes real-world code easier to read.

---

### Function Scope (Global vs Local)

You will learn:

- What global scope is
- What local (function) scope is
- Why some variables are accessible only inside a function

**Why this matters:**  
Scope prevents bugs and keeps code predictable. Knowing where variables exist is essential for debugging and clean architecture.

---

### Hoisting

You will observe:

- Which functions can be used before they are written
- Which ones cannot
- How JavaScript handles function hoisting

**Why this matters:**  
Hoisting explains many confusing beginner errors. Understanding it early avoids frustration later.
